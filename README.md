# Sobe Agency WP

Production-ready WordPress theme boilerplate for Sobe Agency client projects. Built on [Sage 12](https://roots.io/sage/) / Roots Acorn with Tailwind CSS v4, Alpine.js, and native Gutenberg blocks.

## Stack

- **PHP**: Sage v11 Roots Acorn (Laravel container for WordPress)
- **CSS**: Tailwind CSS v4, design tokens in `resources/css/tokens.css`
- **JS**: Alpine.js (interactivity), GSAP + Lenis (animations), React (block editor only)
- **Blocks**: Native Gutenberg — dynamic rendering via Blade, block UI via React
- **WooCommerce**: Side cart, product catalog layouts, customizer controls

## Quick Start

```bash
npm install
composer install
npm run dev
```
